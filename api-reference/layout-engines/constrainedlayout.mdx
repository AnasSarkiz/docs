---
title: '<constrainedlayout />'
---

All children components of a `<constrainedlayout />` will be automatically
laid out by solving for all the `<constraint />`s that are defined within the
`<constrainedlayout />`.

```tsx
export const MyCircuit = () => (
  <constrainedlayout>
    <resistor name="R1" resistance="1k" footprint="0402" />
    <resistor name="R2" resistance="1k" footprint="0402" />
    <constraint pcb xdist="10mm" left=".R1" right=".R2" />
  </constrainedlayout>
)
```
